# Drift Racing Game (Unreal Engine 5.6)

A single-player Blueprint-based drift racing game built in Unreal Engine 5.6.

## Features Implemented
- Drift scoring system
- Lap and checkpoint system
- Garage system
- Custom UI
- Vehicle handling tuning
- Race timer system

## Tech Stack
- Unreal Engine 5.6
- Blueprint Visual Scripting
- Git Version Control

## Current Status
Actively in development.
